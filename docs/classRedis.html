Redis Class Reference

Detailed Description

Redis-for-Arduino client interface.

This is the primary (and currently sole) user-consumable interface class.

It has a constructor that takes a reference to any instance of a Client subclass, allowing it to be used with any communication framework that abstracts the client bytestream in this way (such as the builtin Ethernet class as well as WiFiClient for devices such as the ESP8266).
